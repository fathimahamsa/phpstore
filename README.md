# Recycled Products E-Commerce System

A web-based e-commerce platform built using **PHP** and **MySQL** that enables users to buy and sell recycled products. The system promotes sustainable shopping by connecting sellers offering recyclable or upcycled items with customers looking for eco-friendly products.

## Features

* User registration and login
* Product listing and management
* Buy and sell recycled products
* Shopping cart and order management
* Secure checkout and payment handling
* Order tracking and purchase history

## Tech Stack

* PHP
* MySQL
* HTML / CSS / JavaScript
* Bootstrap

## Workflow

1. User registers and logs in
2. Seller lists recycled products
3. Buyer browses and adds products to cart
4. Order is placed and payment is processed
5. Seller ships the product
6. Buyer tracks order status
